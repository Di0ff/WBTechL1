Вопрос 12
Что выведет данная программа и почему?

func main() {
  n := 0
  if true {
     n := 1
     n++
  }
  fmt.Println(n)
}

Программа выведет 0. Разберем каждую строчку, так как их мало и так более понятно.

  n := 0 // Инициализируем переменную внутри функции main
  if true { // Условие всегда выполняется, так как оно всегда истинно
     n := 1 // Инициализируем новую локальную переменную n внутри блока if
     n++ // Увеличиваем значение этой новой переменной на 1 (теперь n = 2)
  }
  fmt.Println(n) // Выводим значение переменной, объявленной снаружи блока if

Переменная n, объявленная внутри блока if, является новой локальной переменной,
которая существует только внутри этого блока. Поэтому изменение значения этой локальной переменной
не влияет на переменную n, объявленную в основной части функции main.
