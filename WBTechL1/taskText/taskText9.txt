Вопрос 9
Сколько существует способов задать переменную типа slice или map?

Slice

1. Когда мы сразу задаем элементы слайса:
slice := []int{1,2,3}

2. Через make, если нам нужно задать длину и емкость слайса:
slice := make([]int, 3, 3)

3. Вырезать из массива:
array := [3]int{1,2,3}
slice := array[0:2]

4. Пустой слайс:
slice := []int{}
var slice []int


Map

1. Можно сразу задать поля мапы:
m := map[string]int{"один": 1, "два": 2}

2. С помощью make:
m := make(map[string]int, 3)

3. Пустая мапа:
var m map[string]int // nil
m = make(map[string]int) // Инициализация, иначе будет ошибка при взаимодействии с мапой

4. Можно задавать поля мапы в процессе её использования:
m := map[string]int{}
m["один"] = 1
m["два"] = 2 // Но этот подход не самый безопасный вроде как
